   11.URL导向与ThinkPHP的差异是，后者从模块开始向操作解析，Mist则是从操作向模块解析
      优点:URL地址可以很简短，只需要直接输入操作名称就可以访问(在单个模块下优点突出)
      缺点:操作与参数之间的分割符需要着重设置，否则会出现解析歧义
